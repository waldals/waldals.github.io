Intro

Thank you for viewing my profile! Through my 3 years of professional experience with Sungage Financial, 2.0 Group Advisory and Charles River Analytics, (in addition to my studies at Thinkful and McGill University), I have gained valuable experience in the field of data analytics. I learned how to extract, clean and analyze data from various sources to drive business decisions. In this porfolio, I have included personal projects of mine in addition to some coursework capstones which demonstrate my knowledge of SQL, Python, Tableau, Advanced Excel & VBA. If you would like to see more work I have produced for clients and feel I could help contribute to your organization's data needs, feel free to reach out to me! In the meantime, check out my work.

Work

ANALYSIS OF ITUNES LIBRARY IN EXCEL WITH VBA
This personal project of mine is a dashboard built in Excel using VBA that shows tailored statistics of my Itunes Library that no other music platform shows. Feel free to check out how it works and to edit it by following the instructions! I customized a version of this dashboard for 3 companies where I track their KPIs.

AIRBNB ANALYSIS WITH PYTHON
This was the final capstone project for my Thinkful program. I used Python to analyze Airbnb rental units in NYC. Using the language's pandas, visualization and statistical libraries, I identified 3 KPIs to track and see if they affected unit prices. I presented my findings and strategies on how to increase Airbnb's profitability. I used similar analyses in my professional career.

ANALYIS OF HOUSE PRICES USING A/B TESTING IN EXCEL
This was the second capstone project for my Thinkful program. I conducted a statistical analysis of house prices in Excel. I used A/B testing to determine which variables had a correlation on driving up house prices, then presented my findings. I conducted similar work to test for statistical significance when conducting growth experiments.

STOCK DASHBOARD BUILT WITH PYTHON AND XLWINGS
This was the final project for my "Python for Finance Professionals" class. Using Excel's Xlwings module and Python, I built a dashboard that web-scrapes data from Yahoo! Finance and returns it in Excel with a GUI interface.

ITUNES ANALYSIS WITH PYTHON
In this personal project of mine, I conducted data extractions and manipulations of my Itunes Music Library with Python in Jupyter Notebooks. I proceeded to conduct some analysis and generated a report that shows key findings. As I learn more advanced techniques, I update this project!

PYTHON API PROGRAM
This was an exercise in my Thinkful program which involved gathering data from an API. This Python program retrieves the names of all British People who are featured in the Harvard Art Gallery API.

WEB-SCRAPING WITH PYTHON
This was an exercise in my Thinkful program which involved web-scraping. The program scrapes the "Space-News" website with Python and retrieves specific information about its articles.

RENTAL CAR FLEET ANALYSIS
This was one of my first forecast models that I built in Excel, designed to optimize revenue and costs of a car rental company. I learned the basic techniques with my Thinkful program and have since built multiple advanced forecasting models with 2.0 group for various clients.

VISUALIZATION OF STORE DATA
I built this Tableau Dashboard which shows KPIs of stores in the Midwest. While this was one of my projects from my time at Thinkful, I had built similar dashboards at Sungage that tracked our installer partner's KPIs to measure account health.

About

I am a results-oriented business data analyst and program manager with 3 years of professional experience in the non-profit, start-up and governement contracting industries. After entering the solar finance industry shortly after graduating McGill's School of Environment, I turned to the field of data analytics. To date, I have helped over 10 different companies and organizations with various data problems, including, but not limited too: revenue forecasting and modeling , growth experiments, dashboard creation, data analysis for KPI improvement and data extraction. I have experience with Excel, Tableau, SQL, Python and Salesforce and look forward to taking on any new challenge! Feel free to check out my work and to contact me if you have any questions.
