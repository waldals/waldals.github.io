Work

AZURE ML DEMO
This personal project demonstrates how to reproduce a data science workflow in Azure Machine Learning Studio by working with a toy dataset. Here I use a Random Forest Regressor to predict the Average Price of avocados.

NBA STATISTICAL ANALYSIS
This personal project is a data analysis of the 2013-2015 NBA seasons + 2015 Playoffs, including statistical tests on the null hypothesis that LeBron James was not a difference maker (i.e. all players are the same).

CLASSIFICATION ANALYSIS
This personal project used the Titanic Dataset from Kaggle and focuses on machine learning workflow, with the end product being a classification model and various hypertuning methods.

NLP SENTIMENT ANALYSIS
This final project with Lighthouse Labs aimed to assess the sentiment of a consumerâ€™s experience with a product based on interview-style feedback. NLP has advanced significantly since 2018 to assess customer feedback and sentiment towards consumer goods. BERT is a state-of-the-art model that has been developed to obtain insights on this type of data and base models are available to be used as a starting point for new data sets. This project used BERT with customer feedback. The outcome of the model was to be able to assess the sentiment of the feedback. From this, insights into business areas of focus can be obtained.

REGRESSION ANALYSIS
This midterm project with Lighthouse Labs had the goal to predict arrival delays of commercial flights. Often, there isn't much airlines can do to avoid the delays, therefore, they play an important role in both profits and loss of the airlines. It is critical for airlines to estimate flight delays as accurate as possible because the results can be applied to both, improvements in customer satisfaction and income of airline agencies.
